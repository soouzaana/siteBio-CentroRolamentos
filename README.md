# 🚀 Centro dos Rolamentos — Site Bio Institucional

## 📌 Sobre o Projeto

Este projeto consiste em um **site bio** desenvolvido para a empresa **Centro dos Rolamentos**, com o objetivo de centralizar informações rápidas e facilitar o contato direto dos clientes com os vendedores e canais oficiais da empresa.

A proposta foi criar uma página moderna, limpa e funcional, com foco em **usabilidade, performance e experiência mobile-first**, permitindo que o usuário encontre rapidamente:

* Contato direto com vendedores via WhatsApp
* Site oficial da empresa
* Localização da loja
* Informações institucionais rápidas

---

## 🎯 Objetivo

Servir como uma **central de atendimento digital rápida e eficiente**, reduzindo barreiras no contato comercial e melhorando a experiência do cliente ao buscar atendimento, localização ou informações da empresa.

---

## ⚙️ Tecnologias Utilizadas

Este projeto foi desenvolvido com foco em performance e responsividade utilizando:

* HTML5
* CSS3
* JavaScript

Além de:

* SEO básico
* Responsividade avançada
* Otimização de carregamento
* Links externos funcionais
